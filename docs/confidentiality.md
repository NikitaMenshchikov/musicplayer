# Политика конфиденциальности и разрешения

## Обзор разрешений

### Что приложение **не** делает:

- Не отправляет данные в интернет
- Не собирает личную информацию
- Не показывает рекламу
- Не требует аккаунта
- Не использует микрофон или камеру
- Не имеет доступа к контактам

### Что приложение **делает**:

- Работает только с Вашими музыкальными файлами
- Все данные остаются на Вашем устройстве
- Изменения сохраняются только в Ваши файлы
- Работает полностью офлайн

Приложение имеет права на:

- Чтение файлов:

1. Просмотр Вашей музыкальной коллекции
2. Чтение метаданных из MP3 файлов
3. Загрузка обложек альбомов

- Запись файлов:

1. Сохранение отредактированных тегов
2. Добавление обложек в файлы

### Какие данные обрабатываются:

| Тип данных | Где хранится | Использование | Передача |
|------------|--------------|---------------|----------|
| Аудиофайлы | На устройстве пользователя | Воспроизведение |Нет |
| Теги (метаданные) | Внутри аудиофайлов | Отображение, редактирование |Нет |
| Обложки | Внутри аудиофайлов | Отображение |Нет |


## Безопасность


### Что происходит при удалении приложения:

- Ваши аудиофайлы остаются &mdash; они не удаляются
- Все теги сохраняются &mdash; они встроены в файлы
- Обложки сохраняются &mdash; они в файлах
- Библиотка музыки **не** сохраняется. Аудиофайлы необходимо снова загрузить в приложение

### Отсутствие сетевых запросов:

- Нет доступа в интернет &mdash; приложение не запрашивает это разрешение
- Нет серверной части &mdash; всё работает локально
- Нет регистрации &mdash; не нужно создавать аккаунт
