# Основные функции приложения

## Управление библиотекой

### Импорт файлов
В данном приложении треки  загружаются с устройства пользователя. При загрузке треков происходит автоматическое чтение метаданных из файла и отображение информации из них в аудиофайлах

### Организация библиотеки
В приложении на странице альбомов и исполнителей треки группируются по альбомам и артистам соответственно. На странице альбомов в каждом альбоме показана информация о количестве треков в альбоме и сам список треков. На странице исполнителей показана информация о количестве альбомов исполнителя и треков исполнителя. При переходе к конкретному исполнителю показываются его альбомы. При переходе к альбому показывается список треков альбома 

## Редактирование тегов

В приложении можно изменять теги

### Базовые теги
- **Название трека** &mdash; название композиции
- **Исполнитель** &mdash; название исполнителя
- **Альбом** &mdash; название альбома
- **Год** &mdash; год выпуска альбома/трека
- **Жанр** &mdash; музыкальный жанр
- **Номер трека** &mdash; порядковый номер в альбоме
- **Обложка** &mdash; обложка композиции

### Добавление обложек

Обложки для треков загружаются с устройства пользователя. У обложек есть ограничение в 2МБ. Поддерживаемый формат &mdash; JPEG, PNG, BMP


## Воспроизведение музыки

Приложение обладает основными функциями плеера: воспроизведение трека, остановка воспроизведения, перемотка в рамках трека, включение следующего/предыдущего трека. Также имеется автоматическое проигрывание следующего трека, которое можно отключить. Управлять воспроизведением можно через страницу проигрывания трека и через мини-плеер, находящийся внизу страниц
