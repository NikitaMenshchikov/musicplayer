# Решение проблем

## Частые проблемы и их решение

### "Музыка не воспроизводится"

**Симптомы:**
- Трек не начинает играть
- Нет звука
- Ошибка при воспроизведении

**Возможные причины:**
1. Проблема с файлом
2. Проблемы с разрешениями
3. Ошибка в приложении

**Решение:**
1. **Проверьте файл** &mdash; откройте его в другом плеере
2. **Перезапустите приложение** &mdash; закройте и откройте снова
3. **Проверьте громкость** &mdash; не выключен ли звук
4. **Попробуйте другой файл** &mdash; если работает, проблема в конкретном файле

### "Приложение не видит мои файлы"

**Симптомы:**
- Пустой список треков
- Файлы есть на устройстве, но не показываются
- Ошибка "Нет разрешения"

**Решение:**
1. **Дайте разрешения**:
Настройки -> Приложения -> Музыкальный плеер
-> Дополнительно -> Разрешить управление всеми файлами -> Включить

2. **Проверьте файл**:
- Не защищён ли он от записи
- Не находится ли в системной папке
- Есть ли свободное место


### "Не отображаются обложки"

**Симптомы:**
- Вместо обложки стандартная иконка
- Обложка есть в файле, но не показывается
- Ошибка при добавлении обложки

**Решение:**
1. **Проверьте формат обложки**:
- Поддерживаются: JPG, PNG, BMP
- Максимальный размер: 2 МБ

2. **Перезагрузите приложение** &mdash; кэш может быть устаревшим

3. **Добавьте обложку заново**:
- Откройте редактирование тегов
- Выберите "Выбрать изображение"
- Выберите картинку
- Сохраните

### "Приложение медленно работает или вылетает"

**Симптомы:**
- Долгая загрузка
- Подвисание интерфейса
- Неожиданные закрытия

**Решение:**
1. **Очистите кэш приложения**:
Настройки -> Приложения -> Музыкальный плеер
-> Хранилище -> Очистить кэш


2. **Уменьшите библиотеку**:
- Удалите неиспользуемые файлы

3. **Проверьте устройство**:
- Достаточно ли оперативной памяти
- Не перегружено ли устройство
- Нет ли фоновых процессов

4. **Переустановите приложение**:
- Удалите приложение
- Перезагрузите устройство
- Установите заново

## Диагностика проблем

### Проверка файла:
1. Откройте файл в другом плеере
2. Убедитесь, что файл не повреждён

### Проверка разрешений:
1. Откройте настройки приложения
2. Проверьте включённые разрешения
3. Перезапустите приложение

### Проверка устройства:
1. Достаточно ли свободного места
2. Нет ли проблем с памятью


## Восстановление после сбоя

### Если приложение перестало работать:
1. **Не беспокойтесь** &mdash; ваши файлы в безопасности
2. **Перезагрузите устройство**
3. **Очистите кэш приложения**
4. **Переустановите приложение**
