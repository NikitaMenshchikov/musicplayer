# Первые шаги

## Шаг 1: Добавление музыки

### Как добавить файлы:
1. На главном экране нажмите большую кнопку **"Добавить музыку"**
2. Разрешите доступ к файлам (если ещё этого не сделали)
3. Выберите один или несколько аудиофайлов
4. Нажмите "Выбрать"

### Что происходит:
- Приложение копирует информацию о файлах
- Автоматически читает теги (название, исполнитель и т.д.)
- Добавляет файлы в библиотеку
- Показывает уведомление об успешном добавлении


## Шаг 2: Воспроизведение музыки

### Запуск воспроизведения:
1. Найдите нужную композицию в списке загруженных треков
2. Нажмите на него &mdash; он начнёт играть, либо нажмите кнопку воспроизведения напротив трека

### Мини-плеер (внизу экрана):
- Показывает текущий трек
- Есть прогресс воспроизведения
- Управление воспроизведением (пауза, продолжение проигрывания, включение следующего/предыдущего трека)
- Кнопка для открытия полноэкранного плеера


## Шаг 3: Редактирование тегов

### Что можно редактировать:
- **Название трека** &mdash; название трека
- **Исполнитель** &mdash; название  исполнителя
- **Альбом** &mdash; название альбома
- **Год** &mdash; год выпуска
- **Жанр** &mdash; музыкальный жанр
- **Номер трека** &mdash; порядковый номер в альбоме
- **Обложка** &mdash; изображение альбома

### Как редактировать:
1. Нажмите на кнопку редактирования трека
2. Выберите "Редактировать теги"
3. Измените нужные поля
4. Нажмите "Сохранить"

### Важно:
- Изменения сохраняются в сам файл
- Даже если удалить приложение &mdash; теги останутся
- Другие плееры тоже увидят изменения
